# Plantilla de preguntas

## Principios

Las preguntas deben ensenar, no solo medir memoria.

Cada pregunta debe incluir:

- Respuesta correcta.
- Explicacion simple.
- Por que las otras opciones son peligrosas o incorrectas.
- Fuente.
- Tema.
- Nivel.
- Riesgo asociado.

## Tipos de pregunta

| Tipo | Uso |
| --- | --- |
| seleccion_multiple | Examen y repaso rapido. |
| verdadero_falso | Conceptos simples o mitos comunes. |
| escenario | Criterio y conduccion defensiva. |
| imagen_senal | Reconocer senales o marcas viales. |
| ordena_pasos | Tramites, respuesta ante riesgo o procedimiento. |
| reflexion | Conciencia vial y consecuencias humanas. |

## Reglas de redaccion

- No usar trampas.
- No usar opciones casi iguales si confunden sin ensenar.
- Evitar preguntas de memoria pura cuando se pueda usar un escenario.
- Explicar el error con respeto.
- Incluir consecuencias reales cuando el tema lo permita.

## Ejemplo de enfoque

Malo:

"Que articulo regula X?"

Bueno:

"Vas cerca de una escuela y ves ninos en la acera. Que debes hacer primero?"

La segunda pregunta prepara mejor para la vida real.
