# Assets visuales

Esta carpeta documenta imagenes, diagramas y animaciones necesarias para explicar la ley de forma accesible.

## Archivos sugeridos

- `visual-briefs.md`: lista priorizada de imagenes y diagramas.
- `senalizacion.catalog.md`: catalogo de senales necesarias para preguntas.
- `animaciones-3d.briefs.md`: escenas para Three.js o herramienta similar.
- `alt-text.md`: textos alternativos para accesibilidad.

## Principios

- La imagen debe ensenar una decision.
- La senal debe verse clara.
- No usar sangre ni morbo.
- Mostrar consecuencias sin explotar el dolor.
- Usar ejemplos cercanos: redondeles, buses, motos, lluvia, noche, escuelas y paradas.
