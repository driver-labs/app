## Criterio de aprobacion

Una pregunta esta lista cuando:

- Tiene respuesta correcta.
- Tiene explicacion simple.
- Tiene fuente.
- Ensena una decision segura.
- No promete ser "pregunta oficial" si no hay autorizacion.
