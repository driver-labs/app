# Knowledge base de cultura vial SV

Esta carpeta define la estructura de conocimiento para la app. No contiene las lecciones finales; contiene el mapa, reglas y plantillas para que una RAG genere modulos claros desde las fuentes oficiales.

## Objetivo del producto

Ayudar a las personas en El Salvador a aprender la ley de transito en lenguaje simple, practicar para la licencia y, sobre todo, manejar con conciencia. La promesa no es memorizar respuestas: es entender que cada regla protege vidas.

## Fuentes base

- Ley de Transporte Terrestre, Transito y Seguridad Vial.
- Reglamento General de Transito y Seguridad Vial.
- Reglamento General de Transporte Terrestre.
- Presentacion del Manual Centroamericano de Dispositivos Uniformes para el Control del Transito.
- Documento MVP de cultura vial SV actualizado.

## Documentos de estructura

- `00-principios-editoriales.md`: tono, lectura facil, inclusividad y seguridad.
- `01-mapa-de-modulos.md`: taxonomia general y modulos propuestos.
- `02-onboarding-y-rutas.md`: rutas segun nivel y objetivo del usuario.
- `03-especificacion-rag.md`: como chunkear, citar y generar contenido.
- `04-plantilla-modulo.md`: formato base para cada modulo.
- `05-plantilla-preguntas.md`: formato para preguntas, escenarios y explicaciones.
- `06-briefs-visuales.md`: guia para imagenes o placeholders.
- `07-inventario-fuentes.md`: fuentes base, prioridad de autoridad y temas que requieren validacion actual.

## Carpetas

- `modules/`: esqueletos de modulos de conocimiento.
- `question-banks/`: estructura para bancos de preguntas por modulo.
- `assets/`: descripciones de imagenes, diagramas o animaciones.

## Regla central

Cada pieza de contenido debe responder cuatro cosas:

1. Que dice la norma.
2. Que significa en la calle.
3. Que puede pasar si no se cumple.
4. Que decision segura debe tomar la persona.
